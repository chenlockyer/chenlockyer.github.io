|浏览量:

为什么写博客

作为一个机器人方向的学生，日常接触的技术栈很杂：C++、ROS、SLAM、强化学习、仿真……

写博客的目的很简单：

  1. 沉淀知识 — 踩过的坑记录下来，下次不踩
  2. 分享交流 — 开源社区教会我很多，希望能帮到后来的人
  3. 建立品牌 — 让更多人知道我在做什么

会写什么

主要覆盖这些方向：

  • ROS/ROS2：使用经验、源码阅读、工具链
  • **C++**：性能优化、Modern C++ 技巧
  • 机器人：项目实战、仿真训练、Sim2Real
  • 偶尔：读书笔记、技术思考

关于我

深圳技术大学 CS 本科在读，专注机器人系统和强化学习。做过 G1 人形机器人的 RL 训练、远程操控、多机集群等项目。欢迎在 GitHub 上交流。

Let’s build something that moves. 🤖
