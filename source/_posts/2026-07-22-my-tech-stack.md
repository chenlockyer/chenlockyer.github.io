## 编程语言

### Python — 主力胶水语言
- **PyTorch**：强化学习训练
- **OpenCV + NumPy**：图像处理和矩阵运算
- **Flask**：快速搭建机器人控制 API

### C++ — 性能敏感场景
- ROS/ROS2 节点开发
- 实时控制和嵌入式
- CMake 构建系统

## 机器人框架

| 工具 | 用途 |
|------|------|
| **ROS/ROS2** | 机器人中间件，通信框架 |
| **Isaac Lab** | NVIDIA 的 RL 训练框架 |
| **MuJoCo** | 物理仿真引擎 |
| **unitree_ros2** | 宇树机器人 ROS2 驱动 |

## 工具链

- **Git / GitHub**：版本控制和协作
- **Docker**：环境隔离和部署
- **Linux**：日常开发环境
- **VS Code**：主力编辑器
